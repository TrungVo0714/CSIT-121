Web Design Evaluation

Website Information

  • URL: Mathway.com
  • Name: Mathway.
  • Target Audience: Students and/or Teachers.

Site Organization

The site uses a hierachical with two navigation menu on the top left and top right. The top left menu let the users change among different subjects or different level of mathematics; The top right menu provides more information about the website for the users.

CRAP Design Principle

The site uses the Repetition, Proximity, and Alighnment principles. For example, all the buttons represented the numbers and all the mathematics functions are all sitting at the bottom of the website; All the simple functions and numbers are all on the left side and all the more complicated functions are all on the right side. There are gaps and spaces between the buttons and since all the buttons have the same size, it creates rows and columms of numbers and functions.

Navigation Design

  • Dynamic Navigation: Users can click on the top left menu and a columm of different subjects will appear and allows them to change the subject.
  • Site Search Feature: When users click on the top right menu, a search function called Examples appears and allow them to search for different mathematics examples.
  • Site map: The top right menu provides access to more information about the site.

Web Design Best Practices Checklist

Shortcomming(s):

  • Configure frames with frame title and place meaningful content in the noframes area.
  • Captions are provided for each audio or video file used.

Recommendation:

Since Mathway only shows the final answer for a problem and only allow users to see step-by-step answers when they pay for a subscription, having a chat room will be great. A set up like discord will be amazing; where everyone can come in and share information on math problems or other subjects.
